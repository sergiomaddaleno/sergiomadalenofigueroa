Motor Graphics - Second year final project

Motor Graphics - Second year final project
About The Project

A beginner project where I learned the fundamentals of OpenGL. It includes basic rendering techniques like drawing shapes, managing shaders, and handling camera systems in 3D space. The project focused on understanding how to manipulate graphics in real-time using OpenGL.

Features
  • Basic rendering with OpenGL
  • Camera
  • 3D shapes rendering
  • OpenGL buffers and textures
  • Lighting - Directional, spot and point
Technologies Used
  • Language: C++
  • Graphics: OpenGL
  • Utilities: GLFW, GLAD, stb_image
Applications

This project was a stepping stone in learning OpenGL, providing the foundation for more advanced rendering systems. It was an excellent introduction to graphics programming and shader development.

T33Engine - Third year final project

T33Engine - Third year final project
About The Project

T33Engine is a custom-built rendering engine created using modern OpenGL.

Features
  • Shadow Mapping
  • Lighting Systems
  • Geometries
  • Meshes
  • Skybox
  • Volumetric Clouds
  • Camera System
  • Deferred Rendering
  • Screen-Space Ambient Occlusion (SSAO)
  • Postprocessing Effects
  • L-System for procedural generation
  • Fog Effects
  • Entity Component System (ECS)
  • Job System for multi-threading
  • Stencil Outline Effect
  • Scripting Support (Lua)
  • Physics Engine
  • ImGui Interface
  • Transformation gizmo (ImGui)
  • Create, Save, and Load Scenes
  • Hierarchy System
Technologies Used
  • Language: C++
  • Graphics: OpenGL (Modern Core Profile)
  • UI: ImGui
  • Utilities: GLFW, GLAD, stb_image, JSON
  • Other: Lua for scripting, ECS pattern
Applications

This engine served as a base for experimenting with real-time graphics techniques and could be extended as a learning framework or integrated into larger game development pipelines. It supports rapid prototyping and shader experimentation.

Pacman - A* & A+

Pacman - A* & A+
About The Project

This project is a Pacman game developed as an introduction to AI algorithms, specifically A* and A+ search algorithms. The goal is to show how AI can control Pacman’s movements and also how the ghosts behave using pathfinding algorithms.

Features
  • AI-based movement using A* and A+ algorithms
  • Pathfinding for both Pacman and the ghosts
  • Grid-based movement system
  • Basic collision detection with walls
  • Scoring system and game-over conditions
Technologies Used
  • Language: C++
  • Algorithms: A* and A+ for AI pathfinding
  • Input: Keyboard for player controls
Applications

This project serves as an educational tool for understanding how AI algorithms can be implemented in a classic game like Pacman. It helps in learning pathfinding techniques and how AI can be integrated into game development.
