Team33 Engine

Project brief:

T33Engine is a custom-built rendering engine created using modern OpenGL.

Key Features:

  • - Shadow Mapping
  • - Lighting Systems
  • - Geometries (Cube, Sphere, Quad, Surface, Terrain)
  • - Meshes
  • - Cubemap - DDS cubemap
  • - Volumetric Clouds
  • - Camera System
  • - Deferred Rendering
  • - Screen-Space Ambient Occlusion (SSAO)
  • - Postprocessing Effects
  • - L-System for procedural generation
  • - Fog Effects
  • - Entity Component System (ECS)
  • - Job System for multi-threading
  • - Stencil Outline Effect
  • - Scripting Support (Lua)
  • - Physics Engine
  • - ImGui Interface
  • - Transformation gizmo (ImGui)
  • - Create, Save, and Load Scenes
  • - Hierarchy System

  • Utilities - GLFW, GLAD, stb_image, JSON

Low level programming - Assembly

Brief:

Low-level programming in assembly language involves writing instructions that closely correspond to a computer’s machine code. Assembly provides a symbolic representation of the binary commands executed by the CPU, enabling direct hardware control and performance optimization. This approach demands a deep understanding of the processor’s instruction set, registers, memory addressing, and system architecture. Assembly programming is essential in scenarios requiring precise hardware control, speed, and efficient resource management—such as embedded systems, device drivers, and real-time applications.
